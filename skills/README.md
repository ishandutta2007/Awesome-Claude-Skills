# Claude Skills Collection

This directory contains a collection of reusable skill prompts for Claude. Each skill is designed to provide expert-level guidance and structured output for specific development and productivity tasks.

## Structure

Each skill is organized into its own directory:

- `skill.md`: The core prompt instructions, role definition, process, and examples.
- `README.md`: Brief overview and usage instructions for the skill.

## How to Use

1. Navigate to the desired skill directory.
2. Copy the content of `skill.md`.
3. Paste it into your Claude conversation or set it as a system prompt.
4. Provide your specific task or context.
